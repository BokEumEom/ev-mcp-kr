이 문서는 `ev-mcp` (한국환경공단 EV 충전소 정보 MCP 서버) 사용자를 위한 지원 채널과 자주 묻는 질문을 안내합니다. [Claude 원격 MCP 서버 디렉터리 제출 가이드](https://support.claude.com/ko/articles/12922490) 의 필수 문서입니다.

## 1. 사용 시 막히면

다음 순서로 확인해 주세요.

1. **README.md** — 빠른 시작, 환경변수, 7 개 도구 목록과 사용 예시.
2. **`docs/LOCAL_TESTING.md`** — 로컬 개발 환경에서 MCP Inspector 로 도구를 점검하는 절차 (현재 작성 예정, Phase 5-3 에서 추가).
3. **`docs/PHASE3.md` / `docs/PHASE4.md`** — 서버 동작·운영 관련 결정과 알려진 한계.
4. 그래도 해결되지 않으면 아래 **이슈 리포트** 채널로 문의해 주세요.

## 2. 자주 묻는 질문 (FAQ)

### Q1. 주소로 검색했는데 "지오코딩 실패" 에러가 납니다.

`VWORLD_KEY` 환경변수가 설정되지 않았거나 잘못된 키일 가능성이 높습니다. 본 서버는 주소 → 좌표 변환에 [VWorld 지오코더](https://api.vworld.kr) 를 사용합니다.

해결:

- VWorld 에서 무료 키를 발급받아 `VWORLD_KEY` 환경변수로 등록 (Render 대시보드 또는 로컬 `.env`).
- 발급이 어려우면 `find_chargers_nearby` 호출 시 `address` 대신 `lat`/`lng` 를 직접 넘기세요.

### Q2. 도구를 호출하면 응답이 비어 있거나 0 건이 나옵니다.

두 가지 원인이 가장 흔합니다.

- **캐시 워밍 중.** 콜드 스타트 직후 첫 1~2 분은 `station_info` 24시간 캐시가 백그라운드로 채워지는 중입니다. `/health` 의 `station_info.fresh` 가 `true` 가 될 때까지 잠시 대기하세요.
- **`SERVICE_KEY` 가 잘못됨.** data.go.kr 에서 발급한 인증키와 다르면 한국환경공단 API 가 401/사용량 초과 응답을 반환합니다. Render 환경변수 또는 `.env` 의 `SERVICE_KEY` 가 정확한지, 신청한 활용 신청이 승인됐는지 확인하세요. 키 자체는 로그에 노출되지 않으므로 별도 확인이 필요합니다.

### Q3. Claude 가 도구 자체를 못 찾습니다 ("도구가 없습니다" / 연결 실패).

원격 MCP URL 의 **경로** 를 확인하세요. FastMCP 3.x 의 streamable HTTP 경로는 `/mcp` 입니다 (트레일링 슬래시 없음).

- 정답: `https://your-domain/mcp`
- 주의: `https://your-domain/mcp/` 로 보내면 서버가 `307 → /mcp` 로 리다이렉트합니다. POST 본문이 보존되어야 동작하지만, 일부 클라이언트는 307 을 안 따라가므로 정답 URL 을 직접 쓰는 게 안전합니다.
- 오답: `https://your-domain` (경로 누락)

또한 HTTPS 가 활성화돼 있어야 하며 (Claude 는 `https://` 만 허용), CORS 화이트리스트에 `claude.ai` / `claude.com` 이 포함되어 있어야 합니다 (기본 설정에 포함됨).

### Q4. "한국 영역 밖 좌표" 같은 에러가 납니다.

`find_chargers_nearby` 도구는 좌표 입력값이 **한반도 영역**(대략 위도 33.0~38.7, 경도 124.5~132.0) 안에 있는지 검증합니다. 영역 밖이면 `ValueError` 가 발생합니다.

해결:

- 한국 내 좌표 또는 한국 주소를 입력하세요. 본 서버는 한국환경공단 데이터만 다루므로 해외 좌표는 의미 없는 결과만 반환합니다.
- 좌표가 정확한지 의심되면 `address` 인자에 한국어 주소(예: `"서울특별시 강남구 테헤란로 152"`) 를 넣어 보세요.

### Q5. Claude 가 한국어 질문을 잘 이해하지 못하거나 도구를 잘못 부릅니다.

본 서버의 7 개 도구는 모두 **도크스트링이 한국어로 작성**돼 있어 한국어 의도 인식이 잘 되는 모델에서 가장 정확하게 동작합니다.

권장:

- **Claude 4.5 / Claude.ai** 의 최신 모델을 사용하세요. 구버전·소형 모델은 도구 매칭 정확도가 떨어질 수 있습니다.
- 질문을 구체적으로 — "강남역 근처" 보다 "서울특별시 강남구 강남대로 근처 1km 안" 처럼.
- 도구 호출이 안 되면 직접 도구 이름을 언급해 보세요 ("`find_chargers_nearby` 로 …").

## 3. 이슈 리포트 — GitHub Issues

버그·기능 요청·문서 개선은 GitHub 저장소의 Issues 에서 받습니다.

- 저장소 URL: <https://github.com/bokeum/ev-mcp-kr>
- 이슈 트래커: <https://github.com/bokeum/ev-mcp-kr/issues>

> 직접 fork 해서 운영하는 인스턴스를 사용 중이라면 운영자가 안내한 저장소를 우선 확인하세요.

이슈 작성 시 포함:

- 사용한 Claude 클라이언트와 모델 버전
- 호출한 도구 이름과 인자 (개인정보·민감정보 제외)
- 기대한 결과 vs 실제 결과
- 가능하면 `/health` 응답 또는 에러 메시지 (시크릿은 마스킹된 채로 안전)

## 4. 보안 이슈 신고

보안 취약점은 **공개 이슈로 올리지 마시고** GitHub Security Advisory 비공개 채널로 알려 주세요.

- **GitHub Security Advisory**: <https://github.com/bokeum/ev-mcp-kr/security/advisories/new>

`SERVICE_KEY` / `VWORLD_KEY` 노출, RCE, SSRF, 인증 우회, 시크릿 마스킹 회피 등은 모두 위 채널로 부탁드립니다. 신고 시 가능하다면 PoC 코드와 영향 범위를 함께 적어 주세요.

## 5. 응답 시간 기대

- **운영 SLA 가 아님.** 본 서버는 자원봉사·취미 프로젝트이며, 한국환경공단 OpenAPI 와 호스팅 인프라(Render) 의 가용성에 의존합니다.
- 일반 이슈: **영업일 기준 며칠 이내 best-effort** 응답.
- 보안 이슈: 가급적 **영업일 기준 1~2 일 이내** 1차 회신.
- 보장된 응답 시간이나 가동률(SLA) 은 약속하지 않습니다.

서비스 중단·점검 공지가 필요한 경우 GitHub 저장소의 README 또는 Issues 에 공지합니다.

## 6. API 자체 문의 (한국환경공단 OpenAPI)

본 서버는 한국환경공단의 공공 OpenAPI 를 단순 중계할 뿐, 충전소 데이터 자체에 대한 권한이 없습니다. 따라서 다음 사안은 **데이터 출처 기관** 으로 문의해 주세요.

- 잘못된 충전소 정보·좌표·운영시간
- 데이터 누락·오기
- API 인증키(`SERVICE_KEY`) 발급·연장·쿼터
- API 응답 스키마 변경 문의

문의 채널:

- **공공데이터포털**: <https://www.data.go.kr/>
- 데이터셋 페이지: "한국환경공단_전기자동차 충전소 정보" 검색 → 페이지 하단 **문의** 사용

본 서버에서 발견한 문제가 한국환경공단 API 자체의 이슈로 의심되는 경우, 이슈 작성 시 그 사실을 명시해 주시면 분류에 도움이 됩니다.
